# Challenge 5 - Whack-a-Food

Standalone Unity project for the Junior Programmer Challenge 5 exercise.

## What is implemented

- Difficulty buttons that start the round with different spawn rates.
- Random target spawning on a 4x4 board.
- Score updates for clicked food.
- Bad targets end the game when clicked.
- Missed good food ends the game.
- 60 second timer, game over text, and restart button.

Open `Assets/Scenes/Challenge5.unity` in Unity and press Play.
